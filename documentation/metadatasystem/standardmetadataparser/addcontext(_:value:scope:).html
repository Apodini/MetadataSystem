Instance Method

addContext(_:value:scope:)

Adds a new context value to the current context of the MetadataParser.

Declaration

func addContext<C>(_ contextKey: C.Type, value: C.Value, scope: Scope) where C : OptionalContextKey

Parameters

contextKey

The key of the context value

value

The value that is associated to the ContextKey

scope

The scope of the context value as defined by the Scope enum

Discussion

Call this function every time you need to register a new context value for a ContextKey that need to be available for all subsequent Handlers registered in the current Component subtree of the Apodini DSL.
