Protocol

RestrictedMetadataBlock

A RestrictedMetadataBlock is a AnyMetadataBlock which is restricted to only contain a specific Type of AnyMetadata (RestrictedMetadataBlocks support arbitrary nesting, meaning they always can contain themselves).

Declaration

protocol RestrictedMetadataBlock : AnyMetadataBlock

Overview

In order to support multiple AnyMetadata which is allowed inside a RestrictedMetadataBlock, use a class as the base Metadata type, creating subclasses for all allowed Metadata.
